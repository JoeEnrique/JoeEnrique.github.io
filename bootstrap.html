Bootstrap Use

Bootstrap was used as part of this project. But, I intentionally keep it limited. In my personal opinion Bootstrap is an excellent tool. For those that want the job done and don't have time to work with complex coding. Bootstrap is an excellent tool to get your job done.

Because I am an entry-level developer I pass through different ups and downs. I accept that Bootstrap makes it easier to creates grids. This specific page was created following the Bootstrap specifications. But, because my intention was to learn CSS, HTML and JS I avoided using them a lot. I used Bootstrap elements in some cases and others are my own creation with the knowledge acquired from MIT.
